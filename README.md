**Hotel project**  
Практика верстки-учебный проект

1) HTML/CSS/JavaScript, Flexbox Layout, CSS Grid Layout, JQuery 
2) Препроцессор SASS

5) Сборка с помощью Gulp, автоматизация

1) Модальные окна 
2) Анимация
3) Валидация форм 
4) Отправка писем на почту
5) Слайдер
6) Параллакс-эффект
